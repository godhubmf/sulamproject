Toasts / Notifications

Toast notifications are overlay messages that appear in the corner (usually top-right) and auto-dismiss. Use .toast with variant classes.

Toast Container

Toasts should be placed inside a fixed container. Position: top-right by default.

Preview area (toasts would appear in top-right corner)

✓
Changes saved successfully.
✕
Failed to delete item. Please try again.
ℹ
New updates available.
⚠
Session will expire in 5 minutes.
Usage
<!-- Toast Container (fixed position) -->
<div id="toast-container" aria-live="polite" aria-atomic="true">
    
    <!-- Success Toast -->
    <div class="toast toast--success" role="status">
        <span class="toast__icon" aria-hidden="true">✓</span>
        <div class="toast__message">Changes saved successfully.</div>
        <button class="toast__close" aria-label="Dismiss">×</button>
    </div>

    <!-- Error Toast -->
    <div class="toast toast--error" role="alert" aria-live="assertive">
        <span class="toast__icon" aria-hidden="true">✕</span>
        <div class="toast__message">Failed to complete action.</div>
        <button class="toast__close" aria-label="Dismiss">×</button>
    </div>

    <!-- Info Toast -->
    <div class="toast toast--info" role="status">
        <span class="toast__icon" aria-hidden="true">ℹ</span>
        <div class="toast__message">New updates available.</div>
        <button class="toast__close" aria-label="Dismiss">×</button>
    </div>

    <!-- Warning Toast -->
    <div class="toast toast--warning" role="status">
        <span class="toast__icon" aria-hidden="true">⚠</span>
        <div class="toast__message">Session expiring soon.</div>
        <button class="toast__close" aria-label="Dismiss">×</button>
    </div>

</div>
Accessibility Notes
  • Use role="status" with aria-live="polite" for non-critical messages (success, info).
  • Use role="alert" with aria-live="assertive" for critical errors.
  • Add aria-atomic="true" so screen readers announce the entire message.
  • Close button must have aria-label="Dismiss notification".
  • Icon should have aria-hidden="true" (semantic info is in text).
  • JavaScript should auto-dismiss after 3-5 seconds, pause on hover/focus.
Variants
  • .toast--success – Green accent, checkmark icon
  • .toast--error – Red accent, X icon
  • .toast--info – Blue accent, info icon
  • .toast--warning – Orange accent, warning icon
